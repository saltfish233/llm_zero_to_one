## Insight
1. generate中的采样策略非常影响回答质量，原本写的一个简单的采样策略发现回答效果很差，参考minimind的generate采样策略后发现生成质量提升非常大。因此如果模型训练正常的但回答质量不高的话，不一定是模型差劲，也可能是采样策略没写好。
